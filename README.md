HR Analytics Dashboard – Project Description

**Project Title: HR Analytics Dashboard**
Tools Used: Power BI 
**Project Overview:**
Developed an HR Analytics Dashboard to analyze employee data and identify key factors affecting employee attrition, performance, and workforce distribution. The dashboard provides insights that help HR teams make data-driven decisions.

**Key Responsibilities / Work Done**
1.Collected and cleaned HR employee dataset for analysis.
2.Performed data transformation and data modeling to prepare the dataset.
3.Analyzed key HR metrics such as employee attrition, department-wise distribution, job roles, and salary trends.
4.Built interactive dashboards to visualize workforce insights.
5.Used charts and KPIs to track attrition rate, employee count, age distribution, and job satisfaction
